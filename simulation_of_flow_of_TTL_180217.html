Simulation of TTL flow / one dimensional model

2018-02-27



We want to know energy E (Joule) and mass M (Kg) of each cell (volume segment).

where the loop is devided into several cells.

Definition of physical quantities:

The positive direction of the flow is clockwise or right handed.

E: energy (Joule) in cell, M (Kg): mass in cell, v  (meter/s): flow velocity, A (m^2): flow area, V: volume of cell


dx (m): length in direction of flow or width of cell.


dBE (Joule): energy, dBM (Kg): mass, Bv (m/s): velocity,  A (m^2)

 are defined at each cell boundary


dWE (Joule): Calorie come to each cell from wall

 is defined in each cell


T: temperature, rho: density, Cp: specific heat, E: energy, M: mass, V: volume, dx

Updating dE, dBE, dM, dBM:

here, i is an index of each cell (volume segment), t is a time, and dt is the time step.

dE(i) = E(i)_{t+dt}-E(i)_t=dBE(i)-dBE(i+1)+dWE(i)

dBE(i) = (rho(i-1) Cp(i-1) T(i-1))*(v(i) A(i)),     if v(i)>0

             = -(rho(i) Cp(i) T(i))*(v(i) A(i)),    if v(i)<0

dBE(i+1) = -(rho(i) Cp(i) T(i))*(v(i+1) A(i+1)),    if v(i)>0

                = (rho(i) Cp(i) T(i))*(v(i) A(i)),  if v(i+1)<0


dM(i) = dBM(i)-dBM(i+1t,

dBM(i) = (rho(i-1)*(v(i) A(i)), if v(i)>0

         = -(rho(i)*(v(i) A(i)), if v(i)<0

dBM(i) = -(rho(i)*(v(i+1) A(i+1)), if v(i+1)>0

         = -(rho(i+1)*(v(i+1) A(i+1)), if v(i+1)<0
